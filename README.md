# BarberBook — приложение для записи к парикмахеру

PWA-приложение: клиенты записываются к мастеру онлайн, мастера управляют расписанием.

## Возможности

### Клиент
- Список парикмахерских с расстоянием (геолокация)
- Карточки мастеров: уровень, рейтинг, отзывы, прайс-лист
- Запись на свободный слот (30 мин), выбор услуги
- Мои записи, отмена брони

### Мастер
- Расписание по дням, список клиентов с телефонами
- Блокировка слотов и целых дней
- Управление услугами и ценами (взрослые/детские)

## Запуск локально

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Технологии
React + Vite, React Router, Tailwind CSS, PWA, localStorage.
